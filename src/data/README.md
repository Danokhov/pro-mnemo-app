# База ассоциаций - Инструкция по загрузке данных

## Формат данных

Данные хранятся в файле `associationsBase.json` в формате JSON массива.

## Структура записи

Каждая запись должна содержать следующие поля:

```json
{
  "id": "уникальный_идентификатор",
  "Слово": "немецкое слово",
  "Перевод": "перевод на русский",
  "Транскрипция": "[транскрипция]",
  "Часть речи": "глагол | существительное | прилагательное | ...",
  "Темы": ["Тема 1", "Тема 2"],
  "Мнемо якоря": "мнемо-якорь",
  "Ассоциации": "ассоциация",
  "Примеры": "Пример использования",
  "Формы глагола (Inf – Prät – Perf)": "kommen - kam - gekommen", // Только для глаголов
  "Спряжение (Präsens)": "ich komme, du kommst...", // Только для глаголов
  "Родственные слова": "родственные слова",
  "Изображение": "/images/associations/название_файла.jpg"
}
```

## Важные замечания

1. **Поля глагола**: Поля "Формы глагола" и "Спряжение" отображаются только если "Часть речи" = "глагол" или "verb"
2. **Изображения**: 
   - Загрузите изображения в папку `public/images/associations/`
   - Укажите путь в формате `/images/associations/имя_файла.jpg`
   - Поддерживаемые форматы: jpg, png, gif, webp
3. **Темы**: Должны быть массивом строк, например: `["Тема 1", "Тема 2"]`

## Как конвертировать Excel в JSON

1. Откройте файл Excel `База слов 500+.xlsx`
2. Экспортируйте в CSV (Файл → Сохранить как → CSV)
3. Используйте онлайн-конвертер CSV в JSON (например, https://csvjson.com/csv2json)
4. Или используйте Python скрипт для конвертации
5. Замените содержимое файла `associationsBase.json` полученными данными

## Пример Python скрипта для конвертации

```python
import pandas as pd
import json

# Читаем Excel файл
df = pd.read_excel('База слов 500+.xlsx')

# Конвертируем в JSON
data = df.to_dict('records')

# Сохраняем в JSON файл
with open('associationsBase.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)
```
